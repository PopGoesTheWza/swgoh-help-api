SWGoH.help API client for GAS

Google Apps Script client wrapper for the SWGoH.help API

For API access and support, please visit us on Discord

Getting Started

Within you Google document

  1. open the menu Resources and the option Libraries...
  2. add the library by using this ID 1bMXMdwJmhTBxb4tBMEyEtw3jPVTzMZQz6NZ9Wm5-5b2MPWWFfqJVGa6n and selecting the latest version
  3. use the following snippet for simple testing

Build your own copy

Prerequisite:

  • Have Google Clasp CLI installed
  • (Recommended) setup your TypeScript IDE

Steps:

  1. Make a local copy of the GitHub repository.
  2. Edit the file .clasp.json with the scriptId of your own new Google script
  3. If needed, edit the source .ts files under the dist/ directory
  4. Use Clasp CLI to push the transpiled TypeScript into your Google script
  5. (Optional) issue a Pull Request for your changes to be added to the official release
